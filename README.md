## Available Scripts

### `yarn dev`

Starts the the app in development mode.

### `yarn build`

Builds the app for production.

### `yarn preview`

Previews the bundled app (run `yarn build` first).

### `yarn deploy`

Builds the app and deploys it to Cloud Foundry.
